# iZND Group - Interactive Portfolio & Media Kit

This repository contains the digital assets for Prof. Dr. Isaac Govind Andy's personal brand and professional portfolio under the iZND Group. It showcases his trajectory from corporate infrastructure to decentralized intelligence and AI innovation.

## Files Description

### 1. `Portfolio.html`

**The Interactive Portfolio**
This is the main landing page designed for a general audience and potential partners. It provides a highly visual, interactive journey through Prof. Dr. Isaac's career and ecosystem.

* **Key Features:**
  * **Hero Section:** High-impact introduction with profile summary.
  * **Innovation Trajectory:** An interactive timeline switching between Corporate Foundation, Entrepreneurial, and Future Architect phases.
  * **Ecosystem Grid:** Visual cards for major ventures: Kinsight AI, TheAlien.888, iZNDVerse, iZND Coin, and Boardroom Tech.
  * **Professional Matrix:** A dynamic radar chart visualizing expertise distribution (Leadership, AI, Web3, etc.).
  * **WhatsApp Integration:** A floating, interactive WhatsApp button for direct communication.
* **Tech Stack:** HTML5, Tailwind CSS (via CDN), Chart.js, Google Fonts (Space Grotesk & Inter).

### 2. `index.html`

**The Interactive Media Kit**
Designed as a "Digital Portfolio Dashboard," this file is tailored for press, media, and detailed professional inquiries. It offers a more structured, editorial presentation of information.

* **Key Features:**
  * **Dashboard Layout:** Structured sections for "The Architect," "The Legacy," and "The Empire."
  * **Kinsight Logic Simulator:** An exclusive interactive module allowing users to simulate business challenges and receive AI-driven strategic insights.
  * **Detailed Ventures:** Expands on the ecosystem with additional context like "iZND Services" and "iZND Web3 AI."
  * **Speaking & Thought Leadership:** An accordion section detailing key speaking topics (AI Ethics, Blockchain Adoption, etc.).
  * **Visual Style:** Uses "Playfair Display" for a premium, editorial aesthetic.
* **Tech Stack:** HTML5, Tailwind CSS (via CDN), Chart.js, Google Fonts (Playfair Display & Inter).

## Usage

Both files are standalone HTML documents. They can be viewed directly in any modern web browser without a build step.

1. Simply double-click either file to open it in your default browser.
2. Internet connection is required to load external resources (Tailwind CSS, Chart.js, Google Fonts, and images).
